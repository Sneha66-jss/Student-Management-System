# Student Management System

## Description
A console-based Student Management System developed in C++ using file handling and menu-driven operations.

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- File Handling

## Technologies Used
- C++
- File Handling
- Structures
- Functions

## How to Run
1. Compile the program.
2. Run the executable.
3. Use the menu to manage student records.
